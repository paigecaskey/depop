Webscraper for Depop.com

Chromedriver is only updated to Chrome versions 114 and below. If using Chrome 115 or above, you will need to download a newer version of Chromedriver.
Download driver from the google chrome labs website (https://googlechromelabs.github.io/chrome-for-testing/) and directly state executable path in the webdriver. 
Note: Warning will appear asking to pass in a service object when using this method
If using Chrome version 114 and below, Selenium is equipped to automatically find and use the correct version of Chromedriver. 
Currently will not work when using a headless browser
Note that this program automatically accepts cookies
